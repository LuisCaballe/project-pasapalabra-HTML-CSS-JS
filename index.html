Bienvenido a Pasapalabra Game

Cada jugador deberá responder 27 preguntas
en un tiempo límite de 200 segundos.

El jugador podrá usar las siguientes teclas:
· 'Enter' para aceptar.
· 'Control' para posponer la pregunta.
· 'Escape' para terminar la partida o salir.
· 'R' para mostrar el ranking (cuando hayan completado el juego dos o más jugadores).
